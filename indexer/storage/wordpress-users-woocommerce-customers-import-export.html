Import, Export, and Migrate WordPress Users and WooCommerce Customers

This WordPress user import/export plugin lets you import and export your WordPress users and WooCommerce customers from your website. You can seamlessly migrate your user data from one store to another. This will save you from manually adding each user’s data one by one.

Supported file types: CSV, XML, TSV, and Excel

Multiple Methods to Import and Export WordPress Users and Customers

The plugin provides you with multiple options to choose a convenient import/export method that works best for you.

You can import and export the user data using three different methods:

Quick import/export – Import/Export all the data related to the export in one go.

Pre-saved Template – Use a pre-saved template for import/export actions without having to configure the previous settings again.

Advanced import/export – Make use of the advanced import export filters and options to perform custom imports and exports.

Furthermore, you can choose the import file location from:

  • your local storage
  • the URL where the file can be fetched
  • the FTP location of the file

Ensure Secure Migration of User Passwords

The plugin enables you to securely import WordPress user data, including their encrypted passwords. This means you can import user information from another website and maintain the password encryption stored in the WordPress database.

Inform Users about the Data Transfer

When you are migrating your users’/customers’ data to a new website using this plugin, you can inform users about the data transfer via email. This not only keeps you compliant with legal requirements but also ensures your users are aware when their data is moved.

Advanced Filters for Custom Export of User Data

The plugin gives you full control over how you want to export your WordPress users/customers’ data. You can specifically choose to export user data based on their user role, email ID, and registered date.

Apart from the above-mentioned filters, you have options to:
– Choose how many users to export or skip from export
– Sort the columns in the export file the way you like
– Select and rearrange the user data fields you want to export

Bulk Edit or Modify User/Customer Data During Import

With this user import export plugin, you can make bulk updates or edits to existing data on your website. You can use simple mathematical formulae to make changes to WordPress users’/customers’ data

Schedule or Automate Data Import/Export Actions

This plugin allows you to schedule the import/export actions of WordPress users and WooCommerce customers. You can use the scheduler for one-time imports/exports or periodic imports/exports at regular intervals. The plugin allows you to choose the scheduling time and the intervals for the periodic import/export actions.

You can add multiple FTP/SFTP profiles for scheduling the import/exports on your WordPress website. This enables importing or exporting files using multiple FTP profiles without configuring the profiles each time.

View Detailed Log History

This plugin offers easy troubleshooting with developer logs to identify errors in failed import/export actions. It lists all the import/export processes that happened and are scheduled on your website. You can disable or delete unwanted scheduled actions to reduce server load and the chances of the failure of actively scheduled actions.

These debug logs can also be downloaded from your website and can be shared with your developer team for debugging purposes.
